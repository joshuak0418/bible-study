# BEE Korea 귀납적 성경연구 설교 준비 시스템

BEE Korea 「귀납적 성경연구의 지침서 - 성경연구방법 I」 방법론을 기반으로 한
웹 기반 설교 준비 시스템입니다.

## 기능

- **5단계 플로우**: 관찰 → 해석 → 적용 → 설교 구조 → AI 생성·다운로드
- **귀납적 성경연구 전 과정 지원**:
  - 배경 관찰 (저자·장소·시기·수신자·역사적 상황)
  - 문체 관찰 (장르·수사법)
  - 육하원칙 관찰
  - 14가지 구조적 특징 선택
  - 4단계 해석 연구 (내용 조사·문맥 재검토·교차 참조·보조 자료)
  - 20가지 해석 원칙 체크
  - 4영역 적용문 (하나님·자신·타인·세상)
  - Big Idea 및 설교 개요 작성
- **AI 문서 생성**: Claude AI가 연구 내용을 바탕으로 완성된 설교 준비 문서 작성

## GitHub Pages 배포

1. 이 저장소를 Fork 또는 새 저장소 생성
2. `index.html` 파일 업로드
3. Settings → Pages → Source: `main` branch → Save
4. 잠시 후 `https://[사용자명].github.io/[저장소명]` 접속

## 사용 방법

1. **설교 본문** 입력 후 단계별로 연구 내용 작성
2. 5단계에서 **"AI로 설교 준비 문서 생성하기"** 클릭
3. 생성된 초안 확인 후 복사 또는 Claude 채팅에서 Word 파일 생성

## 기술 스택

- 순수 HTML/CSS/JavaScript (의존성 없음)
- Google Fonts (Noto Serif KR, Noto Sans KR)
- Anthropic Claude API
